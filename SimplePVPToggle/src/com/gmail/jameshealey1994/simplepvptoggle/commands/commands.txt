-----------
COMMANDS
-----------
/pvp <on / off> [username] [world]              Changes PVP status for [username] in [world] to <on / off>
/pvp toggle [username] [world]                  Toggles PVP status for [username] in [world]
/pvp setworlddefault [world] <on / off>         Sets default PVP status for [world] to <on / off>
/pvp setserverdefault <on / off>                Sets default PVP status for server to <on / off>
/pvp status [username] [world]                  Displays PVP status of [username] in [world]
/pvp info                                       Displays default PVP status of server and worlds
/pvp reload                                     Reloads config values
/pvp [help]                                     Shows help menu
